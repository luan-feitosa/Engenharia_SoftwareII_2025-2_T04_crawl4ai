# Engenharia_SoftwareII_2025-2_T04_crawl4ai
Projeto para a matéria Engenharia de Software II da UFS que consiste no uso de LLMs para analisar os padrões arquiteturais de um repositório e comparar sua efetividade 

Para rodar o código, simplesmente dê upload do arquivo no Google Collab e substitua o valor de HF_API_KEY de "SUA_API_KEY_AQUI" para sua chave da API do Hugging Face. Então, execute-o normalmente.
